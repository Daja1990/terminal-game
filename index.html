Rules of the Game:


In order to hack the terminal, you need to find the correct word.



Multiple words will be present within the code. If a wrong one is chosen, an infobox will be displayed, showing how many of the words match.
For example, if the correct word is "Bread" and you choose "Fried" 2/5 of the word is correct (R and E are in the same place).



Look for the duds to make it easier - special signs wrapped in either { }, [ ], ( ) or < >. These will replenish life or remove a wrong word.




This is a remake of a mini-game from the game Fallout 3 / New Vegas. Due to the size emulating that of the game, no mobile and/or tablet version is avaliable. Please watch it on a PC for the full experience!
